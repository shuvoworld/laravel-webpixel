# Laravel Starter Kit

A minimal starter kit for Laravel 8 using Bootstrap 5 and Webpixels CSS.

## Features

- Laravel 8
- Laravel UI (for basic scaffolding like authentication)
- Bootstrap 5
- Webpixels CSS (design system for Bootstrap 5)
- Laravel Mix

## Install

```
composer install
```

```
npm install
```

```
php artisan migrate
```

```
php artisan serve
```

```
npm run watch
```
